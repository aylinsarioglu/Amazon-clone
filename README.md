# Amazon Clone

Amazon mobil uygulamasının temel ekranlarını ve kullanıcı akışlarını taklit eden bir React Native projesi.

Bu proje, e-ticaret uygulamalarında sık kullanılan UI yapıları, component mimarisi ve yardımcı fonksiyonların (utils) kullanımını göstermek amacıyla geliştirilmiştir.

## Screens
- Home Screen
- Product Screen

## Features
- Amazon benzeri ana sayfa tasarımı
- Ürün listeleme
- Ürün detay ekranı
- Component bazlı mimari
- Yardımcı fonksiyonlar (utils)
- Dummy data kullanımı
- Temiz ve okunabilir UI

## Tech Stack
- React Native
- JavaScript
- Expo
- React Navigation

## Project Structure
- src/screens  Uygulama ekranları
- src/components  Tekrar kullanılabilir bileşenler
- src/data  Dummy veri yapıları
- src/utils  Yardımcı fonksiyonlar
- src/navigation  Navigasyon yapısı
